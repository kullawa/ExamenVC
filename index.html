Productos y categorías

Agiliza el proceso de venta configurando tus productos o categorías favoritas en la pantalla principal Wally. Todos los productos que pertenecen a una determinada categoría se mostrarán de un mismo color para encontrarlos fácilmente. Puedes incluir modificadores o complementos a tu pedido.
